## 🚀 Step 1: Google Play Console 가입

1. [play.google.com/console](https://play.google.com/console) 접속
2. Google 계정으로 로그인
3. **개발자 등록비 $25** 결제 (한 번만)
4. 개발자 정보 입력

---

## 🚀 Step 2: Production AAB 빌드

터미널에서 실행:
```bash
eas build -p android --profile production
```

## 🚀 Step 4: 스크린샷 준비

### 필요한 스크린샷
- **휴대전화**: 최소 2장, 최대 8장 (1080x1920 권장)
- **7인치 태블릿**: 선택사항
- **10인치 태블릿**: 선택사항

### 권장 스크린샷 구성
1. 홈 화면 (복용 진행률 표시)
2. 약 추가 화면 (카메라 스캔)
3. 약 목록 화면
4. 알림 화면

## ⚠️ 주의사항

1. **건강/의료 앱 정책**
   - 의료 기기나 진단 도구가 아님을 명시
   - 의료 조언을 제공하지 않음을 명시

2. **광고 정책**
   - 테스트 광고 ID를 실제 ID로 교체 필수
   - AdMob 앱 등록 완료 필요

3. **Play Protect 경고**
   - 신규 앱은 초기에 경고가 표시될 수 있음
   - 사용자 수 증가 시 자동 해제
